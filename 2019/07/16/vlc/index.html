使用

添加 Project-Bridge-Header.h 头文件

这里的是 VLC-Bridge-Header.h

  1. 添加 VLC-Bridge-Header.h -> “#import “MobileVLCKit/MobileVLCKit.h””
  2. Build Settings -> Objective-C Bridging Header -> “VLC/VLC-Bridge-Header.h”
